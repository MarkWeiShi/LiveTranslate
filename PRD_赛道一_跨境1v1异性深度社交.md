# PRD｜赛道一：跨境 1v1 异性深度社交（实时翻译视频/语音）

> 产品代号：**LinkU（暂名）— 变现层产品**
> 版本：v1.0（PRD Draft）｜日期：2026-06-15
> 定位（一句话）：因为有了实时语音翻译，你可以和地球另一端的异性，像聊本地朋友一样深聊。
> 战略角色：作为整体产品矩阵的**高 ARPU 变现层**，承接赛道二（兴趣语聊房）外溢的好感关系，并独立承接跨境异性深度社交需求。
> 关联文档：`跨境社交产品调研_Gemini-LiveTranslate.md`、`PRD_赛道二_跨境热点内容语聊房.md`

## 1. 背景与目标

### 1.1 背景
- 跨境异性社交里，**语言不通是第一杀手**；Gemini 3.5 Live Translate（70+ 语种、数秒延迟、保留音色）首次让"母语听到对方声音"成为可能。
- 该赛道已被 Azar/CooMeet 验证商业模型，但现有产品翻译仅 13 语种且体验机械——**我们的技术 wedge 是"翻译质量代差"，而非"有翻译"**。
- 该赛道同时是**杀猪盘高发区 + 性别供需失衡区**，因此本 PRD 把**安全与女性供给治理放在与核心体验同等优先级**。

### 1.2 产品目标（MVP 阶段）
| 目标 | 衡量 |
|---|---|
| 验证"哇时刻"体验 | 首次跨语种视频完成率、翻译后会话时长 ≥ 同语种会话的 70% |
| 验证单位经济 | 单次有效通话毛利 > 0；付费转化 ≥ 行业基准 |
| 验证安全可控 | 可疑账号拦截率、女性 7 日留存、举报率低于阈值 |
| 验证 HelloTalk 导流价值 | HelloTalk 导入用户的真实率、女性供给占比、首会话率 |

### 1.3 非目标（本期不做）
多人直播、复杂公会体系、纯文字社交（弱化）、大规模买量投放。

## 2. 目标人群（Personas）

### Persona A —「探索者·阿哲」（核心付费方 / 需求侧）
- **画像**：28–42 岁男性，中国/中东/东南亚一二线城市，收入中上，有跨文化好奇与异国恋想象。
- **动机**：认识异国异性、体验跨文化浪漫、英语/外语不好但渴望真实交流。
- **痛点**：① 现有交友 App 全是同城同语；② 自己外语差，打字翻译尬聊无温度；③ 不知道对方是不是真人/骗子。
- **付费意愿**：高。愿为"能听到对方母语声音 + 确认真人 + 不限时长"付费。
- **典型场景**：晚上独处时打开 App，想找人视频聊会儿天，最好是异国女生。

### Persona B —「被珍视者·Maria」（核心供给方 / 价值侧）⭐ 最关键
- **画像**：22–32 岁女性，来自语言学习意愿强或跨文化开放度高的地区（如东南亚、拉美、东欧），可能本身是 HelloTalk 语言学习者。
- **动机**：认识外国人、练语言、获得尊重与新鲜社交，部分人有"通过陪聊/收礼物获得收益"的意愿。
- **痛点**：① 被骚扰、被开黄腔、被当成工具；② 安全顾虑（隐私、被截图、被追踪）；③ 难以筛选优质对象。
- **留存关键**：**安全感 + 控制权 + 被尊重**。女性留存是整个产品的生命线——女性走，男性也留不住。
- **典型场景**：受 HelloTalk 邀请尝试，先设置好门槛和偏好，只接受礼貌、认证过的邀请。

### Persona C —「升温者·Kenji」（HelloTalk 漂移用户 / 自然外溢）
- **画像**：HelloTalk 重度用户，和某位语伴长期交流后产生好感。
- **痛点**：HelloTalk 场景偏"学习/正经"，暧昧升温缺乏合适载体；想视频但语言还不够、怕尴尬。
- **价值**：**最高质量、最低获客成本的种子用户**——关系已存在，只差一个深聊载体。

### Persona D —「异国恋·Lina & Tom」（长期关系维系）
- **画像**：已建立跨国情感关系的情侣，分隔两地，语言仍有壁垒。
- **动机**：日常高频视频，翻译辅助消除沟通误差。
- **价值**：极高 LTV、低成本、强口碑；是产品"正向叙事"的代言人，对冲约会软件污名。

## 4. 与 HelloTalk 用户体系的结合（关键差异化）

> HelloTalk 是本产品最大的护城河来源。结合方式 = **账号打通 + 画像复用 + 供给导流 + 信任迁移 + 能力复用**，但**独立品牌**（避免污染 HelloTalk 学习心智）。

| 维度 | 复用方式 | 价值 |
|---|---|---|
| 账号体系 | HelloTalk 一键登录（SSO），可选绑定 | 降低注册门槛、继承身份 |
| 语言画像 | 复用母语/目标语/水平/常用语 | 直接驱动跨语种匹配 |
| 实人/信任 | 继承 HelloTalk 实人认证、信用分、历史举报记录 | **反诈核心**：黑产难以伪装成有学习历史的真人 |
| 女性供给 | 定向邀请 HelloTalk 高质量女性用户做种子 | 解决性别失衡死穴的最佳燃料 |
| 关系链 | HelloTalk 已有语伴关系一键迁移升级 | Persona C 转化 |
| 翻译能力 | 复用术语库 + 叠加 Gemini 实时语音翻译 | 体验与成本协同 |
| 风控/审核 | 跨产品黑名单共享、复用多语种审核团队与模型 | 难以自建的稀缺能力 |
| 支付 | 复用订阅/支付基础设施 | 缩短变现路径 |

**数据合规红线**：跨产品导流与画像复用必须基于用户**明确同意**，遵守各地数据法规；HelloTalk 侧导流以"邀请/授权登录"形式，不做静默数据迁移。

## 5. 核心功能清单（含优先级）

> P0 = MVP 必须；P1 = 留存/变现增强；P2 = 规模化后。

### 5.1 账号与画像
- **F-AC-01（P0）** HelloTalk 一键登录 + 独立注册兜底。
- **F-AC-02（P0）** 画像继承：语言、兴趣、地区、时区、实人认证状态、信任分。
- **F-AC-03（P0）** 意图声明：交友/深聊/语言+社交（用于匹配与合规分流）。

### 5.2 真人与性别认证（供给质量核心）
- **F-ID-01（P0）** 实人活体认证（视频检测）+ HelloTalk 认证继承。
- **F-ID-02（P0）** 女性认证通道（自愿）：认证女性获更高曝光与收益资格、更强安全工具。
- **F-ID-03（P1）** 认证徽章体系（实人/女性/HelloTalk 老用户/高信任分）。

### 5.3 发现与匹配
- **F-MT-01（P0）** 意图+语言+时区+偏好的匹配引擎（复用 HelloTalk 匹配骨架）。
- **F-MT-02（P0）** 卡片/列表发现 + 在线状态（"现在可视频"）。
- **F-MT-03（P1）** 女性侧"反向筛选"：按认证等级/信任分/礼貌分过滤男性邀请。
- **F-MT-04（P2）** 智能推荐（基于历史会话质量而非颜值）。

### 5.4 1v1 实时翻译通话（产品核心）
- **F-CALL-01（P0）** 1v1 视频 + 双向实时语音翻译（Gemini 3.5），保留音色。
- **F-CALL-02（P0）** 实时字幕（原文+译文双语），可开关。
- **F-CALL-03（P0）** **降级策略**：翻译高延迟/失败时自动降为文字字幕，提示对方，避免冷场。
- **F-CALL-04（P0）** 纯语音通话模式（低门槛，女性更安心）。
- **F-CALL-05（P1）** 关键信息二次确认：检测到"金额/转账/外部联系方式/地址"类语义时弹出确认卡（反诈 + 防误译纠纷）。
- **F-CALL-06（P1）** 通话内即时礼物/表情。
- **F-CALL-07（P2）** 实时美颜/虚拟背景（女性供给体验）。

### 5.5 付费与变现
- **F-PAY-01（P0）** 试用额度：新男性用户免费 N 分钟翻译通话（如累计 7 分钟），用完进付费墙。
- **F-PAY-02（P0）** 钻石/分钟充值 + 周/月订阅（解锁无限翻译时长、高级筛选、隐身）。
- **F-PAY-03（P0）** **翻译作为付费点**：免费层=文字字幕翻译；付费层=实时语音翻译+不限时长。
- **F-PAY-04（P1）** 女性收益体系：通话时长/礼物分成、提现，平台抽成（需强合规审查，避免擦边）。
- **F-PAY-05（P1）** 关系订阅（情侣包）：Persona D 长期低价套餐。

### 5.6 安全与反诈（与核心体验同等优先级）🔴
- **F-SF-01（P0）** 行为风控引擎：异常高频邀请、快速索要联系方式/转账、话术模板检测（**作用于翻译后文本**）。
- **F-SF-02（P0）** 一键举报 + 通话内一键退出/拉黑 + 隐藏个人信息。
- **F-SF-03（P0）** 跨产品黑名单（与 HelloTalk 共享）。
- **F-SF-04（P0）** 反诈引导：高风险关键词触发"谨防杀猪盘"提示，禁止站内诱导转账/加密投资话题。
- **F-SF-05（P1）** 女性安全工具：模糊画面预览、录屏检测告警、限制截图、紧急联系人。
- **F-SF-06（P1）** 未成年人保护：年龄认证、可疑未成年识别与拦截。

### 5.7 关系沉淀
- **F-RL-01（P0）** 好友/关注，通话历史，翻译后聊天记录留存。
- **F-RL-02（P1）** 礼物墙、关系亲密度、回访提醒。
- **F-RL-03（P1）** 跨语种异步消息（语音+翻译字幕），衔接两次通话之间。

## 7. 关键流程细节

### 7.1 匹配引擎输入
意图 + 母语/目标语 + 时区重叠 + 偏好（年龄/地区）+ 信任分 + 历史会话质量；女性侧偏好权重更高（反向筛选）。

### 7.2 翻译技术与降级
端到端延迟监控；P95 超阈值或识别失败 → 自动降级文字字幕并提示；关键信息（金额/联系方式/承诺）触发二次确认卡，兼顾反诈与防误译纠纷。

### 7.3 付费漏斗
免费字幕翻译 → 试用语音翻译分钟数 → 付费墙（订阅/钻石）→ 礼物/情侣订阅。**翻译语音始终是付费价值锚点。**

### 7.4 反诈规则引擎（作用于翻译后文本/语义）
高频陌生邀请、快速索要外部联系方式、加密/投资/转账话术、统一模板话术 → 风险分；高分账号限流、女性侧告警、人工复核、跨产品拉黑。

## 8. 指标与验收标准

- **北极星**：周有效跨语种通话数（双方各发言≥N 句且使用翻译）。
- **体验**：首次视频完成率、翻译后会话时长/同语种会话时长比、降级率、延迟 P50/P95。
- **供给健康**：女性 7 日留存、男女活跃比、女性举报率、女性首会话率。
- **变现**：试用→付费转化、付费 ARPU、翻译相关付费占比、**单次有效通话毛利 >0**。
- **安全**：风控拦截率、可疑转账/引流命中率、申诉误杀率。
